# 부스트캠프 웹 풀스택 과정 7기 준비

안녕하세요 부스트캠프 웹 모바일 7기에 웹 분야로 지원한 박하민입니다. 최근까지 저는 주로 자바스크립트와 React 환경으로 프론트엔드 구현하는 공부하고 있었습니다. 하지만 장기적인 개발자 커리어를 가지기 위해서는 지금보다 탄탄한 기초가 필요하다고 생각했습니다. 웹 개발에 대해 전반적인 이해를 가진 개발자가 되기 위해서는 웹 백엔드 또한 알고있는 것이 좋겠다라는 생각도 하게 되었습니다. 그렇기 때문에 웹 풀스택을 다루는 부스트캠프에 지원하게 되었습니다. 또한 특별히 밀도있는 협업을 통해 함께 성장하는 귀중한 경험을 꼭 해보고 싶었기에 정말로 기대하는 마음으로 부스트캠프 7기에 지원하게 되었습니다. 부스트캠프 웹 7기 함격을 위해 준비한 과정을 이 Github 저장소에 기록하였습니다.

### 부스트 캠프 기대효과

- 협동 학습 경험
- 프로젝트 협업 경험
- 프론트엔드, 백엔드 풀스택 경험
- 개발자 역량을 성장시키는 학습 습관 형성
- 기업 네트워킹
- 개발자 동료
- 피드백을 통한 단점 개선
- 탄탄한 기초지식
- 지속 가능한 개발자

- 포기하지 않고 꾸준히 개발자가 되기 위해 노력하고 있음
- 실패를 겪어보았고 이를 통해서 성장하려고 함
- 개발을 좋아함
- 커뮤니케이션을 좋아함
