# 챕터 19. 최소 신장 트리

## 19-1. 최소 신장 트리

최소한의 비용으로 그래프의 모든 정점을 연결하기 위해서는 필요한 간선 외에는 간선들을 제거해 줍니다. 이렇게 모두 연결되고 최소한의 비용으로 연결된 그래프를 최소 신장 트리라고 부릅니다.

### 최소 신장 트리

- 그래프 내의 모든 정점을 포함하는 최소 연결 부분 그래프입니다.
- 신장 트리 중 가중치의 값이 최소여야 합니다.
- 사이클이 발생하면 안됩니다.
- 대표적으로 이번에 배울 크루스칼 알고리즘이 있습니다.

### 크루스칼 알고리즘

- 그리디 개념을 이용해서 구현할 수 있습니다.
- 그래프를 부분 집합으로 분리하고 시작합니다.
- 가장 가중치가 낮은 간선을 선택하고 부분집합을 연결합니다.
- 사이클이 발생하지 않도록 주의 합니다.
- 사이클을 판단하기 위한 알고리즘으로는 Union-Find라는 알고리즘을 이용할 수 있습니다.

### Union-Find 알고리즘

- Union 로직: 서로 다른 두 집합을 합쳐주는 로직
- Find 로직: 집합의 원소가 어떤 집합에 속해있는지 판단하는 연산
- 보통 트리 구조로 구성합니다.
- 편의상 재귀로 구현하는 경우가 많습니다.

최소신장 트리 크루스칼 알고리즘에 대한 강의 영상을 보았지만 아직 정확하게 어떤 알고리즘인지 기억에 남지 않았습니다. 우선 실습을 진행해보면서 조금 더 파악해보도록 하겠습니다.

## 19-2. 최소 신장 트리 섬 연결하기 실습

문제 접근 방법이 잘 떠오르지 않아서 풀이를 보면서 문제 접근 방법을 알아보도록 하겠습니다.

## 19-3. 최소 신장 트리 섬 연결하기 문제 풀이

문제에서 그림이 그래프 형태를 가집니다. 또 최소의 비용, 모든 섬이 서로 통행 가능하도록이라는 키워드를 본다면 최소 신장 트리를 이용해야 한다는 것을 알 수 있습니다. 풀이의 find 함수가 잘 이해가 되지 않아서 인터넷에서 다른 사람의 풀이를 찾아보았습니다. 조금 이해했으나 완벽한 이해는 하지 못했습니다. 완벽한 이해는 다음 이해력 컨디션이 좋을 때 다시 보고 이해하면 좋을 것 같습니다.
